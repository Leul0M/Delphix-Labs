&gt; **Run your own AI agent locally. Private, secure, and fully under your control.**

Delphix Labs is a lightweight, extensible AI agent framework that runs entirely on your machine. Chat with it via Telegram, give it tasks, let it read files, execute commands, and automate your workflow - all without sending your data to the cloud.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🔒 **100% Local** | Runs on your hardware, data never leaves your machine |
| 🤖 **Telegram Integration** | Chat with your agent from anywhere via Telegram |
| 🛠️ **Tool System** | Extensible tools for file operations, shell commands, and more |
| 🧠 **Multiple Models** | Support for Qwen, Llama, Mistral, and other Ollama models |
| 🎨 **Beautiful CLI** | One-command installer with colored output |
| 🏗️ **Modular Architecture** | Easy to extend with new capabilities |
| 📊 **Conversation Memory** | Maintains context across messages |
| 🛡️ **Built-in Safety** | Sandboxed execution and permission controls |
